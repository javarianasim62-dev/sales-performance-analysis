# Sales Performance Analysis Project

## Overview
This project analyzes sales data to evaluate business performance, identify revenue trends, and understand product and regional sales behavior.

## Objectives
- Analyze overall sales performance
- Identify top-performing products
- Understand sales trends over time
- Evaluate regional performance

## Tools Used
- Python
- SQL (via Python)
- Power BI
- Excel

## Project Structure
- python_sql/ → data cleaning, processing, and SQL analysis
- data/ → raw dataset
- powerbi/ → interactive dashboard
- excel/ → supporting files
- images/ → dashboard screenshots

## Analysis Performed
- Data cleaning and preprocessing using Python
- SQL queries for sales aggregation and insights
- Trend analysis for revenue and sales growth
- Product and region-wise performance analysis

## Key Insights
- Top products contribute the majority of revenue
- Sales show trends over time (monthly/weekly patterns)
- Certain regions perform better than others
- High-performing segments drive business growth

## SQL Usage
SQL queries were executed within Python scripts to analyze and aggregate sales data.
